Creating a DataReader

A DataReader is added to a Subscriber by default when a Subscriber is created. Additional DataReaders can be created by selecting a DataReader from the palette and adding it to the DataReader compartment of a Subscriber. QoS policies can be added to the DataReader as needed.

The DataReader must be connected to a Topic, ContentFilteredTopic, or MultiTopic to be semantically complete. See Connecting Figures for information on how to connect a DataReader to a topic related figure.
